About the Algorithm
Why Dijkstra Matters
Edsger W. Dijkstra published this algorithm in 1959. It remains one of the most fundamental algorithms in computer science.
Algorithm Steps — Click Animate
  • 01Set all node distances to ∞, except source = 0
  • 02Add all nodes to a min-priority queue (unvisited set)
  • 03Extract the node with the minimum tentative distance
  • 04For each unvisited neighbor, compute tentative distance
  • 05If tentative distance < known distance, update it
  • 06Mark current node as permanently visited
  • 07Repeat steps 03–06 until destination is reached
  • 08Backtrack via previous-node map to reconstruct shortest path
Complexity
Time: O((V+E) log V)
Space: O(V)
No negative weights
Always optimal
